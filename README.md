
# Codepulse

Codepulse is a project with a Go backend and a planned Vite frontend.

## Backend

The backend is written in Go.

### Usage

To start the backend, run:

```bash
go run main.go
```

## Frontend

A frontend built with Vite is planned for this project. Stay tuned for updates!
